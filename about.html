About Online Law System

This project is executed especially for this purpose. The project online law system is a software system that contains all the laws of IPC so that people can come and search for the law for respective crime. The laws are categorized into different sections according to some keywords. There is also search option available where user can find or search according to his requirement by entering his query. There is also a facility available for users to get contact of lawyers. An enquiry form is also provided where user can ask their query and can also register case and the form is sent to the court authority. This project also contains information of all the respective courts in area for respective crimes. This online system is developed on .net platform and supported by a Sql database to store user specific details
